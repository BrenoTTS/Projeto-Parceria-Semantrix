# Projeto-Parceria-Semantrix
um Projeto com finalidade de Resolver ou mitigar problemas da vida real com Analise de dados, nesse projeto eu usei uma base de dados publicos, do kaggle com dados sobre o disperdicios de alimentos no mundo, dos anos de 2018 até o ano de 2024, vendo a relação de pais que mais desperdiçou e quais alimentos foram, qual ano teve mais desperdicios .
